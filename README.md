# リバーシサバイバー

リバーシの「挟んでひっくり返す」ルールを、サバイバー系アクションに組み込んだブラウザゲームです。プレイヤーは移動しながら白石を配置し、敵が広げる黒石を反転させてライン攻撃と爆発で敵を倒します。

## 特徴

- `index.html` だけで動作する、依存関係なしの HTML5 Canvas ゲーム
- PC とスマートフォンの両方に対応
- WASD / 矢印キー、または画面スワイプで移動
- リバーシ判定による反転ライン攻撃
- 経験値、レベルアップ、アップグレード選択
- フィーバーゲージによる一時的な無敵・高速化
- 一定時間ごとに発生するボス戦

## 遊び方

1. `index.html` をブラウザで開きます。
2. 「ゲームスタート」を押します。
3. プレイヤーを移動させ、足元に白石を配置します。
4. 黄色く光るマスに入ると、黒石を挟んで反転攻撃が発生します。
5. 敵を倒して経験値ジェムを集め、レベルアップ時にスキルを選びます。
6. HP が 0 になるまで生存時間と到達レベルを伸ばします。

## 操作方法

| 操作 | 入力 |
| --- | --- |
| 移動 | `W` `A` `S` `D` または矢印キー |
| スマートフォンで移動 | 画面をスワイプ |
| レベルアップ選択 | 矢印キー / `W` `S` |
| 決定 | `Enter` または `Space` |

## ゲームシステム

### 陣地

プレイヤーは移動中に白石を置きます。敵は移動しながら黒石を広げます。白マスの上ではプレイヤーの移動速度と回復量が上がり、敵は弱体化します。

### 反転攻撃

リバーシと同じように、白石で黒石を挟めるマスが攻撃チャンスです。反転可能なマスは黄色く光り、そこへ移動すると挟んだ黒石が白石に変わり、反転ライン攻撃と爆発ダメージが発生します。

### レベルアップ

敵を倒すと経験値ジェムが出現します。経験値が一定量に達するとゲームが一時停止し、以下のようなアップグレードから選択できます。

- 自動配置の間隔短縮
- 反転ダメージ強化
- 爆破範囲拡大
- 移動速度上昇
- 経験値吸引範囲拡大
- 最大 HP 増加と全回復

### フィーバー

黒石を反転させるとフィーバーゲージが溜まります。満タンになると一定時間フィーバー状態になり、無敵化、移動速度上昇、周囲の白マス化が発生します。

### ボス戦

一定時間ごとに `DUEL BOSS` が出現します。ボス撃破後は経験値ジェムが多く出現し、次のボスはさらに強化されます。

## 開発・実行

このリポジトリにはビルド手順やパッケージマネージャ設定はありません。ローカルで確認する場合は、ブラウザで直接 `index.html` を開いてください。

PowerShell 7 で簡易サーバーを使う場合の例:

```powershell
pwsh -NoLogo -NoProfile
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000/` を開きます。
